## Usage

To create a new project base on the Workleap starter kit, run:

### npm

```cmd
npm create @foundry-cli/workleap <your-app-name>
```

### yarn

```cmd
yarn create @foundry-cli/workleap <your-app-name>
```

### pnpm

```cmd
pnpm create @foundry-cli/workleap <your-app-name>
```
