## Usage

To create a new project base on the Workleap starter kit, run:

### npm

```cmd
npm create workleap <your-app-folder>
```

### yarn

```cmd
yarn create workleap <your-app-folder>
```

### pnpm

```cmd
pnpm create workleap <your-app-folder>
```
